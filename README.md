# 极简博客系统

 技术栈：FastAPI+Vue3+SQLite

部署要求：2核2G服务器

介绍：

这是一个基于轻量化原则设计的个人内容发布系统，旨在通过架构的解耦实现资源利用率的最大化，特别适合在计算资源受限的环境下部署。 

该系统采用前后端分离的模块化设计，将数据处理与交互展示进行物理层面的拆分，从而显著降低了并发请求对系统内存的占用。其核心优势在于高响应速度与低维护成本，通过自动化的内容编排机制，简化了从内容生产到终端呈现的链路。用户仅需通过简易的部署流程即可完成系统构建，并能通过标准化的工作流实现内容的快速发布与同步更新。 

整体而言，这是一套兼顾性能表现、运维效率与扩展灵活性的个人数字化内容展示解决方案。
